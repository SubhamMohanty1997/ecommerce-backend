# E-Commerce Backend (Microservices)

Production-ready e-commerce backend built using:
- Java 17
- Spring Boot 3
- Microservices Architecture
- PostgreSQL
- Redis
- RabbitMQ
- OAuth2 + JWT
- AWS (RDS, S3, CloudWatch)

## Services
- auth-service
- product-service
- user-service
- order-service
- inventory-service
- notification-service
- api-gateway
